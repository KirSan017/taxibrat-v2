02

Путь водителя

Воронка из 6 шагов: от бесплатного привлечения до монетизируемого контакта. Первые 3 шага без авторизации для максимального охвата.
1

Hook (лендинг)

Строка поиска «Введите название парка», топ-3 скрытых парка. Никаких барьеров — чистое привлечение.

Конверсия: посетитель → заинтересованный пользователь
Без авторизации
2

«Горькая правда»

Текущий рейтинг парка (например 3.1/5), список слабых параметров. Шок-контент, мотивация к действию.

Эмоциональный триггер через неудовлетворённость
Без авторизации
3

Каталог

Фильтрованный список, названия скрыты («Парк #12»), рейтинги видны. Интрига нарастает.

Есть лучшие парки — но нужен доступ
Без авторизации
4

Регистрация + верификация

Telegram-логин, загрузка фото водительского удостоверения + селфи. Ключевой барьер.

Фильтрация нецелевых — только реальные водители
Барьер
5

Одобрение

Менеджер проверяет данные, бот отправляет контакты парка через Telegram.

Момент монетизации: контакт = лид
Авторизация
6

Чат с куратором

Прямой Telegram-чат с менеджером для сопровождения.

Повышение конверсии лида в трудоустройство
Авторизация
03

Путь парка

Два пути попадания парка в каталог: органический (инициатива платформы) и платный (инициатива парка).

Тайный покупатель проводит аудит ▶ менеджер заполняет 40+ полей ▶ рассчитывается рейтинг ▶ парк в каталоге

Парк не знает о присутствии. Аудит инициирован платформой.

Парк регистрируется ▶ заполняет профиль ▶ аудитор проверяет ▶ выбирает тариф ▶ получает лиды

Парк активно заинтересован. Платит за размещение и лиды.
