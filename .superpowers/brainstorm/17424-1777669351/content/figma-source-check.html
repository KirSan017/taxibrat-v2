Проверка источника Figma для главной

В проекте уже есть локальный референс figma-home-ref.png. Нужно подтвердить, что именно он является эталоном для pixel perfect.

A. Использовать локальный Figma-референс

Сравниваем текущую главную с этим PNG и правим только визуальный слой.

B

Нужна живая Figma-ссылка

Если PNG устарел, пришлите URL файла или конкретного node. Тогда снимем актуальный контекст из Figma.

C

Другой эталон

Можно взять другой локальный файл или скрин из Figma, если главная должна соответствовать не этому изображению.
